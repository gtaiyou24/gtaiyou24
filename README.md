### 🕸️ Epic Bot - 汎用型クローリングシステム

| リポジトリ | 説明 |
|:---------|:-----|
| [page-downloader](https://github.com/gtaiyou24/page-downloader) | Webページをダウンロードするシステム |
| [epic-crawler](https://github.com/gtaiyou24/epic-crawler) | Webサイトをクローリングし、htmlページをダウンロードするシステム |
